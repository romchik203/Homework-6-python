# Working Time Algorithm

Этот файл содержит извлеченный алгоритм расчета рабочего времени по городским
графикам. Алгоритм учитывает только рабочие часы и исключает выходные.

## Что внутри

- `WorkingTimeCalculator` с методами для расчета рабочего времени в часах.
- Настраиваемые графики по городам в `CITY_SCHEDULES`.
- График по умолчанию в `DEFAULT_SCHEDULE`.

## Как пользоваться (интерактивный режим)

1. Запустите файл:

```
python working_time.py
```

2. Введите город и даты в одном из поддерживаемых форматов:

- `2026-01-19 10:30`
- `19.01.2026 10:30`
- `2026-01-19T10:30`

Если оставить город пустым, будет использован график по умолчанию.

3. Программа выведет:

- время реакции (от направления до начала работ);
- время решения (от начала до завершения работ);
- общее время (от направления до завершения работ).

## Пример использования (код)

```python
from datetime import datetime
from working_time import WorkingTimeCalculator

start_dt = datetime(2026, 1, 19, 10, 0)
end_dt = datetime(2026, 1, 20, 12, 0)

hours = WorkingTimeCalculator.calculate_working_time(
    start_dt, end_dt, city="Москва"
)
print(hours)
```
